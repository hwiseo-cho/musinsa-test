## MUSINSA Backend Engineer (Core Biz Platform 파트너서비스/플랫폼팀) 과제

### 개발 환경
- Language : Java
- JDK : 17
- DateBase : H2
- Build Tool : Gradle

### 실행 및 테스트 방법
- 실행 
    -     ./gradlew build
    -     java -jar ./build/libs/musinsa-1.0.jar --
- 테스트 방법
  - 프로그램 실행 시 자동으로 DB schema.sql, data.sql 실행하여 테스트 환경 구축
